Frontend разработка

Самый популярный

Научитесь создавать современные веб-интерфейсы с нуля

Этот курс идеально подходит для начинающих. Вы освоите HTML, CSS, JavaScript и React - технологии, которые используют крупнейшие компании мира.

Что вы изучите:

  • HTML5 и семантическая верстка
  • CSS3, Flexbox, Grid, анимации
  • JavaScript ES6+, асинхронность
  • React, hooks, state management
  • TypeScript основы
  • Работа с API и REST
  • Git и GitHub

Ваши проекты:
